ZEROLORE - QUICK PLAY RULES

Use any miniatures. Agree on a points value. Start playing.

These rules are for getting started quickly and simply. Add more rules whenever you want.

Game size

Before starting, agree on the total points value per player.

Recommended:

  • 50-100 points value -> Skirmish game
  • 150-250 points value -> Squad game

Points value represents:

  • The unit's cost.
  • Its strategic weight when controlling objectives.

You can play in:

  • Individual mode (each miniature is one unit).
  • Squad mode (multiple miniatures form one unit).

Setup

  1. Place 3 objectives on the table.
  1. Deploy in opposite zones.
  1. Determine who starts.

Turn structure

Players activate one unit at a time, alternating turns.

When a unit activates, it can perform 2 different actions:

  • Move - 1 action
  • Shoot - 1 action
  • Charge - 2 actions
  • Prepared - 2 actions
  • Melee combat - 2 actions

It cannot repeat the same action.

When all units have been activated, the round ends.

Movement

The unit can move up to its MOV value in inches (").

Squads must maintain coherency according to the basic rules.

Shooting (Ranged attack)

Every ranged attack is resolved using this sequence:

  1. Choose target

    Choose a visible enemy unit within weapon range.

  1. Determine attacks

    Roll as many dice as the weapon's Attacks value.

  1. Roll hits

    For each die: if the result is equal to or higher than Hit, it scores a hit.

    A result of 6 is a critical hit.

  1. Roll saves

    The defender rolls as many dice as hits received.

    Each result equal to or higher than Save blocks one hit.

    Only a defensive 6 can block a critical hit.

  1. Apply damage

    Each unblocked hit deals the weapon's listed damage.

    Normal hits deal base damage.

    Critical hits deal the weapon's critical damage, if it has one.

When a unit loses all its Lives, remove it from the game as destroyed.

Prepared

A unit can choose the Prepared action.

If it becomes the target of a shot before its next activation:

  • It may immediately make a return shot against the attacker.

First resolve the enemy attack.

Then resolve the return shot.

Charge

To start melee combat, a unit must perform the Charge action, which costs 2 actions.

  • Roll 2D6: that result is the total distance it can move toward the target.
  • If it does not reach the target, the charge fails and the unit remains in place.
  • If the charge is successful, place its base in contact with the target and it is considered engaged.
  • A successful charge automatically includes a melee attack.

Melee combat

It occurs when two units are in contact.

If a unit charged, it attacks first; otherwise, the unit with higher Speed attacks first.

Combat is resolved as follows:

  1. Attack; only 1 and 2 count as failures.
  1. Saves
  1. Damage
  1. Counterattack (as long as the unit still has actions).

If a unit loses all its Lives, it is removed. If both survive, they remain engaged until one falls.

Objectives and victory

The game lasts 4-5 rounds.

At the end of each turn:

  • Add up the value of your units in contact with each objective.
  • The player with higher VAL controls that objective.

The winner is the player controlling more objectives and scoring the most points.

General rule

If a rule does not appear here, it is not used in Quick Play.

For full rules, consult the official rulebook.
