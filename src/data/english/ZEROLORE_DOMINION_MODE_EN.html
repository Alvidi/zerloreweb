ZEROLORE - DOMINION MODE

Domination Mode is ZeroLore's most strategic format, inspired by great real-time strategy games. Battles don't start with entire armies — they start with a vanguard, and the outcome depends on how you manage your resources and control the territory.

Preparation

Players agree on the game size and consult the value table:

Large60 pts200 points8/turn15/turn20/turn

Each player receives tokens worth equal to their total bank. The initial value is deducted from the bank from the first moment — in a Standard game you start with 30 pts already spent and you have 70 pts left in the bank.

Each player builds their first wave with the starting value available. In squad mode units do not have to be assembled beforehand — you can create and configure your squad on the fly with available points.

Command positions are placed on the table where the players agree. It is recommended to use different colors to differentiate the types of positions and their points. Each player has 1 Main Post in its deployment area that belongs to it from the beginning.

During the game

At the end of each turn each player receives value tokens for each command post they control, deducting them from their bank. When the bank is empty, no more tokens are received even if posts are controlled.

Those points can be spent at any time on:

  • New units — deployed from your deployment area or from a command post you control. The cost is the value of the unit according to the army generator.
  • Command doctrines — add new doctrines to your army during the game at the following cost:

Fight to the Death
Declare this doctrine during the Initiative Phase by designating an allied unit that is engaged in close combat. Both units are engaged in a dead-end duel. Neither of them can withdraw or be separated from the fight for any reason. At the start of each turn, before any other activation, both units automatically resolve a close combat exchange following the normal attack order rules. The combat ends only when one of the two units is destroyed. This doctrine can only be applied to units in close combat. Does not affect vehicles or titans.4 pts

New units enter the table at the beginning of the next initiative phase, at a command post or in the deployment zone.

Victory

If a player captures the opponent's Main Post, they win immediately. While they control it, that post keeps generating tokens from their bank.

If no one takes the Main Position, at the end of the last turn whoever controls the most command positions wins. In the event of a tie, whoever has eliminated the most value of units wins.

Duration

Minimum 5 shifts. The players agree on the exact number before starting.
